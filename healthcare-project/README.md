Healthcare Data Analysis (SQL Project)

**Project Overview**
This project focuses on analyzing hospital patient visits, doctor performance, and treatment cost trends using SQL. The goal is to extract meaningful insights from healthcare data that can help hospitals improve patient care, operational efficiency, and cost management. The analysis is performed using multiple relational tables representing hospital operations such as patients, doctors, appointments, treatments, and billing records.

**Objectives**
1.Evaluate treatment costs across departments
2.Identify top-performing doctors
3.Measure hospital readmission rates

**SQL Concepts Used**
This project demonstrates several intermediate to advanced SQL techniques, including:
1.JOINs for combining multiple healthcare datasets
2.Aggregations (SUM, AVG, COUNT) for cost and patient analysis
3.Window Functions (RANK, LEAD, LAG) for advanced analytics
4.Views to simplify complex queries
5.Stored Procedures for reusable analysis queries
6.Date Functions to analyze trends over time.

The list of queries used to perform above mentioned concept are available in healthcare_project folder and named as healthcare_project.sq
