**E-Commerce-Analytics-SQL:**
SQL-based E-commerce Analytics Project E-Commerce Analytics Using SQL

**Project Overview:**
Performed end-to-end data analysis on the Olist Brazilian E-Commerce Dataset (available on Kaggle) to analyze customer behavior, sales trends, and revenue performance. The project focused on transforming raw transactional data into actionable business insights using advanced SQL techniques.

**Dataset Tables:** 
  1.customers 
  2.orders 
  3.order_items 
  4.products 
  5.payments

**Tools Used:** MySQL Workbench

**SQL Concepts Applied: **
  1.Multi-table JOINs 
  2.GROUP BY & HAVING 
  3.Window Functions (RANK, DENSE_RANK) 
  4.CTE (Common Table Expressions) 
  5.Subqueries 
  6.Date Functions 
  7.Customer Churn Analysis 
  8.Cohort Analysis

**Business Problems Solved:** 
  1.Monthly Revenue Trend 
  2.Top Selling Categories 
  3.Customer Lifetime Value (CLV) 
  4.Repeat vs One-time Customers 
  5.Churn Customer Detection 
  6.Payment Method Analysis

For the details of queries used for this project please refer to queries.sq file in the project folder.
