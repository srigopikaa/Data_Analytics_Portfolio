**HR Analytics**

**Project Overview**
This project focuses on analyzing employee attrition, salary trends, and workforce dynamics using SQL.
The goal is to uncover patterns in employee behavior and provide actionable insights that help organizations improve retention, 
compensation strategies, and promotion planning.
**Objectives**
1.Identify factors contributing to employee attrition
2.Analyze salary growth trends across departments and roles
3.Evaluate promotion patterns and gaps
4.Understand the impact of attendance and experience on employee retention
**Dataset**
Source: IBM HR Analytics Dataset (Kaggle)
The dataset includes employee-level information such as demographics, job roles, salary, and work history.
**The project is structured using the following tables:**
1.employees → Employee demographics and job details
2.departments → Department-level information
3.salaries → Compensation history
4.promotions → Promotion records
5.attendance → Employee attendance and absence tracking
**SQL Concepts Used**
1.Window Functions
2.PARTITION BY for segment-wise analysis
3.LAG / LEAD for trend and change detection
**Advanced Analytics**
1.Percentile functions for salary distribution
2.Ranking functions for performance comparison
**Recursive CTEs**
1.To analyze hierarchical or time-based trends
**Key Analysis & Insights**
1.Identified departments with high attrition rates
2.Analyzed salary progression trends over time
3.Detected employees with long promotion gaps
4.Evaluated correlation between attendance and attrition
**Advanced Use Cases (Expert Level)**
1.Attrition Risk Prediction
2.Identify employees likely to leave based on historical patterns
3.Salary Growth Analysis
4.Track salary increments across years and roles
5.Promotion Gap Analysis
6.Measure time between promotions and detect stagnation
**Tools Used**
SQL Workbench
**Conclusion**

This project demonstrates how SQL can be used beyond basic querying to perform advanced analytics and business decision-making.
It highlights the ability to transform raw HR data into meaningful insights that can drive strategic workforce planning.
